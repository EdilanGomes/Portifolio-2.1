Minha Principal Área de Estudo

HTML5

eu uso o HTML5 para criar a estrutura básica das páginas da web. Com ele, defino o layout, organização e o conteúdo das páginas, como títulos, parágrafos, listas, links e imagens.

CSS3

eu uso o CSS3 para dar estilo e formatar as páginas da web. Com ele, posso criar layouts mais complexos e aplicar efeitos visuais sofisticados, como sombras, transições e animações.

JavaScript

Com ele, consigo adicionar interatividade e dinamismo às páginas da web. Utilizo o JavaScript para criar funcionalidades como animações, validações de formulários, carregamento dinâmico de conteúdo e muito mais.

BootStrap

Com ele, consigo criar interfaces de usuário responsivas de forma super rápida. Uso o Bootstrap para agilizar o desenvolvimento, já que oferece uma tonelada de componentes e estilos prontos.

GitHub

o GitHub é uma ferramenta essencial no meu dia a dia. Utilizo o GitHub para armazenar e gerenciar meus projetos de código-fonte, colaborar com outros desenvolvedores e controlar as versões do meu trabalho.

IA

Utilizo-a para aprimorar minhas habilidades técnicas e otimizar processos de desenvolvimento. Embora não dependa exclusivamente dela, recorro à IA para tirar dúvidas, obter certos dados e até mesmo automatizar tarefas quando necessário.
